WGame Layer Documentation

Use layers for drawing stuff.

Classes

Level Layer (abstract)

This is the base class of all layers.

  • constructor - does nothing
  • draw(camera) - displays stuff based on the camera position

BackgroundImageLayer

Use this layer to display larger images. Maybe for background and foreground layer. It's features are:

  • Repeat image
  • Scroll speed can be x times faster/slower than camera
  • Add more images to the layer to let it animate

Structure:

  • constructor(image, relativeSpeed = 1, animationSpeed = 33) - Give it am image and maybe you like to costumize the relativeSpeed to camera or the animationSpeed if you add more images.
  • addImage(image) - Add image for animation

Sprite layer

All movable tiles with special animations like player, enemy should use the sprite layer. It depends on sprite objects.

  • constructor - nothing
  • addSprite(sprite) - add a sprite to the layer

Tile layer

More or less static parts of the layer which are stored in a two dimensional grid.

  • constructor(tileWidth, tileHeight, tileImage, tiles) - You must specify the dimension of a tile, an image which holds the tiles and a tiles object.
