WGame Documentation

WGame is a HTML5 game engine so I can finally finish my next game on Ludum Dare. It's written in CoffeeScript and provides some helpful features you need for game development like sprite animation, layers, level editor...

WGame is published under the Lesser General Public License (LPGL) so you have the right to use it for non-free projects.

Modules

  • Levelformat: Explains the structure of the level file.
  • Layers: All supported layers and how they work.
  • Level: How the level object works.
  • Action: How to connect trigger and code.
