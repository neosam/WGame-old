WGame Level file format

The levelfile format of WGame is based on a json file which links to all the ressources which are needed.

Structure of level 0.1

  • type: always "WGame Level"
  • version: in this case "0.1"
  • tilesImage: path to the image which holds all the tiles
  • width, height: level dimension in sprites
  • tiles: one dimensional array which holds the tiles. A tile holds:
    • index: the tile index on the tile image.
    • visible (optional): true if visible, false if not, default is false.
    • walkable (optional): if a creature can walk on this field. default is false
  • layers: array of layers. Different kinds of layers are:
    • BackgroundLayer: a scrollable animated image layer
      • name: Layer name
      • type: "BackgroundLayer"
      • animationSpeed: when the next frame should be displayed
      • relativeSpeed: scrolling speed relative to camera
      • imagePaths: path to all the images in an array
    • TileLayer: holds the tiles
      • name: Layer name
      • type: "BackgroundLayer"
    • SpriteLayer: holds the sprites
      • name: Layer name
      • type: "BackgroundLayer"
